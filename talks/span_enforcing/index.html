Theme: Diagnosing the gapless modes

What is macroscopic (IR) physics
given a microscopic (UV) description / model, and vice versa?

?

1 : gapless (e.g. conductor)

: gapped (e.g. insulator)

The theme of this talk is: starting from a microscopic (UV) model—interacting particles, atoms, spins—we ask what we can already say about the macroscopic (IR) behavior without solving everything. Can we tell in advance whether it will behave more like a conductor or an insulator? More precisely, we focus on the gap: the IR phase is gapless when there are excitations with arbitrarily small energy, as in a conductor, and gapped when there is a finite energy gap, as in an insulator. This is a fundamental question in theoretical physics that arises in many contexts.

RG Flow and Symmetry Constraints

  • Microscopic symmetry \(\mathcal{S}_\text{UV}\) acts on the IR theory \(\mathcal{T}_\text{IR}\) via \(\phi_{\text{RG}}\)

💡 Utilize \(\mathcal{S}_\text{UV}\) to constrain possible IR physics!

💡 Utilize \(\mathcal{S}_\text{IR}\) to engineer potential UV model!

Tentative:

Let me spell out the picture in the diagram. The top squiggly arrow is the renormalization group flow: starting from the microscopic theory \(\mathcal{T}_\text{UV}\) we coarse‑grain toward an effective long‑distance description \(\mathcal{T}_\text{IR}\). Simultaneously, any symmetry present in the UV does not disappear; instead there is a canonical map of symmetry data \(\phi_{\text{RG}}: \mathcal{S}_\text{UV} \to \mathcal{S}_\text{IR}\). Concretely, UV symmetry operators can be pushed forward along RG and still act on IR observables (sometimes factoring through a reduced or emergent symmetry).

Why is this useful? If we know \(\mathcal{S}_\text{UV}\) and how it can or cannot act on a gapped topological phase, we can often rule out certain IR possibilities. In particular, when the UV symmetry plus its anomaly (or more subtle categorical structure) cannot be realized by any symmetric TQFT, the IR cannot be fully gapped—this is the guiding philosophy behind “symmetry constraints” and the span criterion we will introduce. The punchline: symmetry information flows “downwards” via \(\phi_{\text{RG}}\) and gives us a filter on allowed IR phases before doing detailed dynamics.

Gapped Phases and Symmetric TQFTs

  • Gapped phases (with a nice thermodynamic limit) : TQFTs
    • \(\mathcal{T}_\text{IR} \in \mathrm{TQFT}(\mathcal{S}_\text{UV})\) : symmetric TQFT
  • Mathematically:
    • \(\mathrm{TQFT}(\mathcal{S}) \cong \mathrm{Mod}(\mathcal{S})\) : cat. of over \(\mathcal{S}\)
    • Classification of possible gapped phases

Example: Finite Invertible Symmetry \(\mathrm{Vect}_H^\omega\)

  • An \(H\)-gapped phase Gauging \((K ,\alpha)\)
    • \(K \stackrel{i}{\hookrightarrow} H\) : subgroup, \(d\alpha = i^\ast \omega\) : discrete torsion
    • In 1+1d: Module category: \(\mathrm{Vect}_{H/K}^{\alpha}\)
  • Physics:
    • \(H \rightsquigarrow K\) : SSB, stacked with \(K\)-SPT \(\alpha\)
    • Vacua: \(\ket{gK}\) for \(gK \in H/K\)

Example: \(\mathrm{Vect}_{\mathbb{Z}_2}\)

  • Two gapped phases
    • Full SSB (\(\mathrm{Vect}_{\mathbb{Z}_2}\)) : \(K = \{e\}\), \(\alpha = 0\) : vacua \(\ket{\uparrow}, \ket{\downarrow}\)
    • Trivial (\(\mathrm{Vect}\)) : \(K = H\), \(\alpha = 0\) : vacuum \(\ket{\mathrm{sym}}\)
  • \(\mathrm{TQFT}(\mathbb{Z}_2) \cong \mathrm{Mod}(\mathrm{Vect}_{\mathbb{Z}_2}) \cong \mathbb{N}[\mathrm{Vect}_{\mathbb{Z}_2}, \mathrm{Vect}]\) 1
    • \(\ni \mathrm{Vect}_{\mathbb{Z}_2}^{\oplus n_1}\oplus \mathrm{Vect}^{\oplus n_2}\)
    • vacua : \(\ket{\uparrow}_a, \ket{\downarrow}_a\) (\(a=1,\ldots,n_1\)),   \(\ket{\mathrm{sym}}_s\) (\(s=1,\ldots,n_2\))

  1. It is more precise to write \(\mathrm{2Vect}[\mathrm{Vect}_{\mathbb{Z}_2}, \mathrm{Vect}]\), etc. In higher dimensions this would be crucial.

Example: \(\mathrm{TY}(\mathbb{Z}_2)\)

  • \(\mathrm{TY}(\mathbb{Z}_2) \ni \eta, \mathsf{D}\) : fusion category for Kramers-Wannier duality
    • \(\eta \otimes \eta \cong \mathbf{1}\),   \(\mathsf{D} \otimes \eta \cong \mathsf{D}\),   \(\mathsf{D} \otimes \mathsf{D} \cong \mathbf{1} \oplus \eta\)
  • Only one simple module/ TQFT : \(\mathrm{TY}(\mathbb{Z}_2)\) itself : Full SSB phase
    • vacua : \(\ket{\mathbf{1}}, \ket{\eta}, \ket{\mathsf{D}}\)
    • \(\eta \ket{\eta}\) = \(\ket{\mathbf{1}}\),   \(\mathsf{D} \ket{\mathsf{D}} = \ket{\mathbf{1}} + \ket{\eta}\), etc.
  • \(\mathrm{TQFT}(\mathrm{TY}(\mathbb{Z}_2)) = \mathbb{N}[\mathrm{TY}(\mathbb{Z}_2)] \ni \mathrm{TY}(\mathbb{Z}_2)^{\oplus n}\)  
    • \(\ket{\mathbf{1}}_{j}, \ket{\eta}_{j}, \ket{\mathsf{D}}_{j}\)   ( \(j=1,\ldots,n\) )

Restricting to Sub-Symmetry

  • \(\mathrm{Vect}_{\mathbb{Z}_2} \stackrel{i}{\hookrightarrow} \mathrm{TY}(\mathbb{Z}_2)\) : generated by \(\eta\)
  • Pullback functor \(i^* : \mathrm{TQFT}(\mathrm{TY}(\mathbb{Z}_2)) \to \mathrm{TQFT}(\mathbb{Z}_2)\)
    • \(\eta\ket{\eta} = \ket{\mathbf{1}}\),   \(\eta\ket{\mathsf{D}} = \ket{\mathsf{D}}\)
    • \(i^*(\mathrm{TY}(\mathbb{Z}_2)) = \mathrm{Vect}_{\mathbb{Z}_2} \oplus \mathrm{Vect}\) = (\(\mathbb{Z}_2\)-SSB) \(\oplus\) (\(\mathbb{Z}_2\)-trivial)
    • KW duality \(\mathsf{D}\) exchanges \(\mathbb{Z}_2\)-SSB phase and \(\mathbb{Z}_2\)-trivial phase.

\(i^*(\mathrm{TQFT}(\mathrm{TY}(\mathbb{Z}_2))) = \mathbb{N}[(\mathrm{Vect}_{\mathbb{Z}_2} \oplus \mathrm{Vect})]\) \(\subsetneq\) \(\mathrm{TQFT}(\mathbb{Z}_2)\) \(\ni (\mathrm{Vect}_{\mathbb{Z}_2} \oplus \mathrm{Vect})^{\oplus n}\)

Symmetry Span

The span \(\mathcal{D} \leftarrow \mathcal{E} \rightarrow \mathcal{C}\)

  • Two symmetries \(\mathcal{C}\) and \(\mathcal{D}\) sharing a common sub-symmetry \(\mathcal{E}\)
  • Further embedded into a bigger symmetry \(\mathcal{S}^\text{faith} \curvearrowright \mathcal{T}\)
  • The universal one (pushout) is unfamiliarly big (e.g. free product);
    • \(\mathcal{S}^\text{faith}\) depends on \(\mathcal{T}\)

Span-Symmetric TQFTs

If \(\mathcal{T}_\text{IR}\) is a TQFT, \(i_\mathcal{C}^\ast \mathcal{T}_\mathcal{C} \cong i_\mathcal{D}^\ast \mathcal{T}_\mathcal{D} \in \mathrm{TQFT}(\mathcal{E})\)

Or,
\(i_\mathcal{C}^\ast \mathrm{TQFT}(\mathcal{C}) \cap i_\mathcal{D}^\ast \mathrm{TQFT}(\mathcal{D}) \cong \{0\}\) no symmetric TQFT!

Excludes TQFT phases; does not rule out exotic non-TQFT gapped phases (e.g. fractons in \(d>2\)).

Focused 1+1d setup

  • \(\mathrm{TQFT}(G) \cong \mathbb{N}[G\mathrm{-SPTs}]\) 1
    • \(i_\mathcal{D}^* \mathrm{TQFT}(G) \subset \mathbb{N}[H\mathrm{-SPTs}]\)

  1. In 1+1d relativistic systems, the Mermin-Wagner theorem even forbids SSB of \(G\).

Self-dual \(\mathrm{U}(1)\)-\(\mathbb{Z}_N\) Clock Chain

  • \(\mathcal{H} = \bigotimes_j \mathbb{C}^N_j\)
  • \(X^N = Z^N = 1\) , \(XZ = e^{2\pi i/N} ZX\)
  • \(\mathsf{D}\) : \(X \leftrightarrow Z^\dagger Z\)
  • \(Q \propto \sum_j \sum_{n\neq 0}^N \frac{X_j^n}{1 - e^{2\pi i n /N}}\)
    \(Q' \propto \sum_j \sum_{n\neq 0}^N \frac{(Z_j^\dagger Z_{j+1})^n}{1 - e^{2\pi i n /N}}\) Onsager algebra

    • \(H_n = i\sum_{j,a} \frac{1}{1-\omega^{-a}}\left[(2a-n)X_j^a + n(S_j^+ S_{j+1}^-)^{n-a} - n(S_j^- S_{j+1}^+)^a\right]\)
